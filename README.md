#  Авторизация с Firebase

Приложение с системой аутентификации на основе Firebase Authentication. Пользователи могут работать с приложением без авторизации, но некоторый функционал доступен только после входа в систему.

## Описание

Проект демонстрирует реализацию полноценной системы авторизации с использованием Firebase Auth. Приложение предоставляет:
- Регистрацию и вход по email и паролю
- Публичный контент, доступный всем пользователям
- Закрытый функционал, доступный только авторизованным пользователям
- Возможность выхода из аккаунта

##  Функциональные возможности

### Для всех пользователей:
- Просмотр публичного контента
- Регистрация нового аккаунта
- Вход в существующий аккаунт

### Для авторизованных пользователей:
- Доступ к премиум-функционалу
- Просмотр личных данных
- Выход из аккаунта (Logout)
